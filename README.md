# Character Storyboard Generator

> 作者: QQ奶茶大神


从动画角色图片或真人照片生成透明背景的 Q 版动作与表情分镜 Skill。

- 保留 30 种姿势与 40 种表情，可进行交叉组合。
- 真人照片先转换成一致的 Q 版角色。
- 未成年人及年龄不明角色强制启用非性化保护。
- 输出为透明背景 PNG，并采用简单的“情绪-表情”资料夹分类。
- Sample 当前提供 31 张实际生成范例，包括自然站立的 18 种表情。

- 新手请先看 [使用说明（小白版）](./使用说明-小白版.md)。
- 进阶请阅读 [完整教学](./教学.md)，或直接查看 [透明背景范例](./Sample/成果)。

> Sample 中的角色图片仅用于工作流程示范，不代表对原角色形象授予任何再发行或商业使用许可。
